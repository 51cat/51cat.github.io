## Problem

```
Problem
A common substring of a collection of strings is a substring of every member of the collection. We say that a common substring is a longest common substring if there does not exist a longer common substring. For example, "CG" is a common substring of "ACGTACGT" and "AACCGTATA", but it is not as long as possible; in this case, "CGTA" is a longest common substring of "ACGTACGT" and "AACCGTATA".

Note that the longest common substring is not necessarily unique; for a simple example, "AA" and "CC" are both longest common substrings of "AACC" and "CCAA".

Given: A collection of k (k≤100) DNA strings of length at most 1 kbp each in FASTA format.

Return: A longest common substring of the collection. (If multiple solutions exist, you may return any single solution.)
```

问题：
`input`: 对于给定的`fasta`
`output`: 所有序列的最长的共有序列(`length >= 2`), 多于一条的时候，输出一条即可

```
# input:
>Rosalind_1
GATTACA
>Rosalind_2
TAGACCA
>Rosalind_3
ATACA
# output
AC
```
思路：

- 读入所有序列后找出最短序列后，根据最短序列递减长度去寻找`Motif`
## solution -1
```python
class Motif:
    def __init__(self, fa):
        self.fa = fa
        self.seq_all = None
        self.short_seq = None
        self.motif = None
    
    def parse_fa(self):
        with open(self.fa) as fd:
            for line in fd:
                if not line.startswith(">"):
                    yield line.strip("\n")
    
    def add_use_data(self):
        self.seq_all = sorted([seq for seq in self.parse_fa()],
                              key = lambda x: len(x), 
                              reverse=True)
        
        self.short_seq = self.seq_all.pop()
    
    def find_motif(self):
        for k in reversed(range(0, len(self.short_seq))):
            for i in range(len(self.short_seq) - k + 1):
                query = self.short_seq[i:i + k]
                is_have = [seq.find(query) for seq in self.seq_all]
                
                if -1 not in is_have:
                    self.motif = query
                    return
        self.motif = -1
   
    def get_motif(self):
       return self.motif
   
    def run(self):
        self.add_use_data()
        self.find_motif()

def main():
    fa = "D:\\Desktop\\fa.txt"
    test = Motif(fa)
    test.run()
    print(test.get_motif())

if __name__ == '__main__':
    main()
```
